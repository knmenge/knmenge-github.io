Project 1: Alarm Clock

This project is an alarm clock application built using HTML, CSS, and JavaScript. It allows users to set timers, alarms, and view a real-time clock. Additionally, users can enter the name of a city to display the current time in that location, using a time zone API to fetch the data. The interface includes modals for setting alarms and timers, with options to specify hours, minutes, and seconds. This project demonstrates my skills in creating interactive user interfaces, handling asynchronous API calls, and managing state in a web application. By using JavaScript, I implemented the logic for real-time updates, alarm scheduling, and time zone conversions. This project showcases my ability to integrate third-party APIs, manage dynamic content updates, and create a seamless user experience.

Project 2: Language Probability Checker

This project is a language probability checker that determines whether a given text is more likely to be in English or Spanish. Built using HTML, CSS, JavaScript, and Python with Flask, it allows users to input text and click a button to check the language probability. The python backend is hosted using Render. text is sent to the backend where it is analyzed based on the frequency of each letter compared to typical frequencies in English and Spanish. It returns the probabilities of the text being in each language, which are then displayed to the user. This project demonstrates my ability to create interactive web applications, handle server-client communication, and perform basic text analysis. It highlights my skills in using JavaScript for sending requests and updating the webpage, and Python with Flask for processing data on the server. This project showcases my ability to integrate front-end and back-end technologies to create a functional and user-friendly application.

Project 3: Flappy Bird Game

This project is a recreation of the popular Flappy Bird game using HTML5 Canvas and JavaScript. Players control a bird navigating through gaps in pipes, featuring a scoring system, sound effects, and a mute/unmute button. The game demonstrates my understanding of gravity, collision detection, and score tracking. I used event listeners to handle user inputs and the HTML5 Canvas API for smooth graphics and animations. Audio feedback enhances the user experience, and the interface includes a start button, game-over modal, and a mute/unmute button. This project showcases my skills in creating interactive web applications, handling real-time user input, and implementing game logic.
